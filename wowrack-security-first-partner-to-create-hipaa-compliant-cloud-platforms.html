Wowrack, Security First partner to create HIPAA-compliant cloud platforms

The companies announced the collaboration today at the Healthcare IT News Big Data and Healthcare Analytics Forum in San Francisco.
Wowrack, a provider of cloud-based hosting platforms and colocation services, has partnered with Security First, a software-defined data security technology provider, the companies announced at the HIMSS and Healthcare IT News Big Data and Healthcare Analytics Forum held in San Francisco.

Wowrack and Security First will launch three new HIPAA-compliant private cloud platforms.
[Also at Big Data and Healthcare Analytics Forum: Atul Butte: Precision medicine makes doctors nervous]

The Security First technology brings cryptographic splitting capabilities to Wowrack’s cloud storage service. Security First platform is government-certified to secure classified data and has Federal Information Processing Standard certification.
Further, Wowrack’s cloud storage and database is designed to meet compliance and regulatory standards for providers with robust database and security needs.
Security First CEO Mark O’Hare said the new offerings will store, safeguard and make sure health data complies with HIPAA.

“The desire to use cost-effective cloud storage has grown tremendously with the exponential growth of data in the digital age,” Wowrack chief revenue officer Ed Kimm added. “The healthcare industry in particular has enormous amounts of highly sensitive data that needs to be protected, as well as meet stringent compliance requirements.”
The new platforms are a more cost-effective tool for organizations looking for a storage option with the highest level of encryption, the companies said.
